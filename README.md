# SQL Aggregation and Group By Practice

This repository contains SQL queries for practicing **Aggregation Functions, GROUP BY, and HAVING clauses** using the Olympic Athlete dataset.

## Dataset
The dataset contains information about Olympic athletes including:

- Name
- Sex
- Age
- Height
- Weight
- Team
- Sport
- Year
- Season
- Medal

---

# Query Categories

## 1. Moderate Level (Basic Aggregations)

Practicing basic SQL aggregation functions:

- COUNT()
- AVG()
- MIN()
- MAX()
- SUM()

Examples:
- Count total athletes
- Find average athlete age
- Find maximum height
- Find minimum weight
- Count female athletes
- Count summer season participants
- Count distinct sports
- Find youngest and oldest athlete

---

## 2. Intermediate Level (GROUP BY)

Practicing grouping data by categories.

Examples:
- Number of athletes per sport
- Average age per sport
- Maximum height per sport
- Athletes per country
- Average weight by gender
- Minimum age per Olympic year
- Medal count per country
- Average height by season
- Maximum weight per sport
- Athletes per Olympic year

---

## 3. Advanced Level (GROUP BY + HAVING)

Filtering grouped data using HAVING.

Examples:
- Sports with average age > 30
- Teams with more than 1000 athletes
- Sports with more than 500 athletes
- Olympic years with more than 2000 athletes
- Teams with average height > 180
- Sports with max weight > 120
- Countries with more than 100 medals
- Sports with average weight > 80
- Olympic years with more than 100 medals
- Teams with average age > 28
